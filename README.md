# NeuroFlow Web (Demo)

Демо-веб-приложение на Next.js (App Router) + TypeScript + TailwindCSS для сценария:

`скан страницы -> Focus Reader -> мини-тесты -> финальный тест`

## Как запустить

1. `npm i`
2. `npm run dev`
3. Открыть `http://localhost:3000`

## Что используется в демо

- Нет логина и авторизации.
- Нет реального AI.
- Импорт текста и кнопка "Сгенерировать" используют локальный demo JSON.
- Все данные и прогресс хранятся в `localStorage`.

## Страницы

- `/` — Home (старт + описание режимов ADHD / Dyslexia)
- `/import` — импорт текста / загрузка демо
- `/reader` — Focus Reader по секциям
- `/section-test?i=0..n-1` — мини-тест секции
- `/final-test` — финальный тест (10 вопросов)
- `/results` — история попыток и очистка истории

## Доступность

- Настройки размера шрифта
- Настройки межбуквенного интервала
- Высокий контраст
- Поддержка клавиатуры и skip-link к основному содержимому
## Данные

- Demo JSON: `src/data/temo.json` (скопирован из пользовательского файла `temo.json`)
